# /write-note — Note記事生成

レース予想結果からNote記事を生成します。

## 使い方

```
/write-note [race_id]
```

- `race_id` 省略時はサンプルレースを使用

## 実行手順

1. **前提確認:**
   - 予想結果が存在するか（`output/json/{race_id}.json`）
   - 存在しない場合は `/race-predict` を先に実行

2. **記事生成パイプライン実行:**
   ```bash
   source .venv/bin/activate
   keiba lead
   # メニュー [7]「記事生成」を選択（Agent 13-16を実行）
   ```

3. **結果確認:**
   ```bash
   cat output/markdown/{race_id}.md | head -100
   ```

4. **品質チェック:**
   - 禁止表現17種が含まれていないか確認
   - 無料/有料境界が正しいか確認
   - JRA-VAN出典表記があるか確認
   - 免責事項が含まれているか確認
   - QA スコア 100/120 以上であることを確認

5. **ユーザーへ結果報告:**
   - 記事タイトル
   - 文字数
   - 禁止表現チェック結果
   - QA スコア
   - 出力ファイルパス

## 成果物

- `output/markdown/{race_id}.md` — Note記事（無料5セクション＋有料13セクション）
- `output/eda/{race_id}/` — EDAチャート（5種PNG）

## 禁止表現（必ずチェック）

絶対、確定、鉄板、確実、必勝、必ず儲かる、回収保証、100%、稼げる、儲かる、必ず当たる、間違いなく、間違いない、これだけ買えば勝てる、負けない、ノーリスク、安全
